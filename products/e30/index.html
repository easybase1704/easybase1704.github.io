云终端 · X86平台

E30

支持 Intel®  Apollo Lake平台系列处理器；

  • 支持 英特尔® Apollo Lake平台系列处理器；
  • 最大支持4GB内存，支持DDR3L-1866MHz内存技术；
  • 提供HDMI接口 显示接口，支持4K显示;
  • 最大支持10个USB接口（2 x USB接口 3.0+8 x USB接口 2.0） ;
  • 支持mSATA 固态硬盘、EMMC5.0 Flash存储模式;
  • 支持M.2插槽（2230）扩展无线/蓝牙；
  • 提供1个千兆以太网络接口
  • 采用无风扇静音散热设计;

详细规格

系统 处理器英特尔® 赛扬N3350平台处理器

存储 EMMCMax. 64GB
mSATAFull Size Mini-PCIe Slot For mSATA

音频 1 x 3.5mm 麦克风， 1 x 3.5mm耳机
指示灯1 x 硬盘指示灯, 1 x 电源指示灯
管理功能 网络唤醒
