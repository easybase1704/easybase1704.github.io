EB16

EB16产品是一款高性能、风扇散热结构的边缘计算准系统，机箱使用钣金材质；

  •  
  • EB16产品是一款高性能、风扇散热结构的边缘计算准系统，机箱使用钣金材质
  • 2*DDR4内存 
  • SODIMM内存插槽,支持2133MHz内存,最大支持32GB
  • ，支持 可选HDMI接口/DP接口/VGA接口显示输出,最大支持2个千兆网卡，支持网络唤醒、PXE功能
  • 支持widows和Linux系统，支持英特尔® OpenVINO AI开发套件

详细规格

系统 处理器英特尔® 12th/13th 处理器
芯片组 H610
固件 AMI 固件
内存2*SO DIMM DDR4内存 最大支持 32GB
显示 显卡 英特尔® HD 集成显卡
HDMI接口 1 x DP接口/HDMI1.4b

 VGA接口 1 x VGA接口 / HDMI1.4b
扩展 M.2插槽1 x M.2插槽 2230 扩展 无线+蓝牙
存储 M.2插槽1 x M.2插槽 2280 扩展固态硬盘（SATA/PCIe协议 ）
I/O 接口以太网Realtek 8111H;
1x RJ45（可选2 x RJ45）
USB接口4 x USB3.0；
4 x USB2.0（双网口时，2 x USB2.0）
音频 1 x 3.5mm 麦克风， 1 x 3.5mm耳机
指示灯 1个电源指示灯，1个硬盘指示灯
串口 可选1 x RS232
管理功能 网络唤醒
上电自启
看门狗
电源 类型直流输入
电压 19V
功率 65W / 90W
软件支持支持运行windows10、Linux OS；
支持英特尔® OpenVINO AI开发套件；
环境工作温度 0 ℃ ~ 40 ℃
存储温度 -20 ℃ ~ 60 ℃
存储湿度 95%@40 ℃（无冷凝）

尺寸183mm x 189mm x 37mm

认证 CE/FCC(DoC) Class B，CCC
